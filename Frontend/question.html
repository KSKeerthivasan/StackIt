What is the difference between HTML and HTML5?

Can someone explain the key differences between HTML and HTML5 in terms of new features and compatibility?

▲ 12 ▼

Answers

HTML5 introduces new elements like <video>, <audio>, and <canvas> that were not present in HTML. It also supports local storage, offline apps, and semantic tags like <article> and <section>.

Unlike HTML, HTML5 supports drag and drop, new form input types, and better error handling in browsers.
